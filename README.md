# Chikoo
Javascript library for browser storage and event management.

## Installation
Install via NPM:

```bash
npm install chikoo

```

## Usage

#### javascript

```javascript

var myApp = require("chikoo");

myApp.printMsg(); // print and return " Hello World! " 

```

#### TypeScript

```typescript

import * as myApp from 'chikoo';

myApp.printMsg(); // print and return " Hello World! " 


```
